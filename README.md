# NEXUS | Premium Tech E-Commerce Dashboard

NEXUS is a sleek, modern front-end e-commerce dashboard built to demonstrate strong foundational web development skills. It features a "glassmorphism" aesthetic, dynamic product filtering, and a fully functional shopping cart using local storage.

## ✨ Features

- **Premium UI/UX:** A responsive, dark-themed design with smooth micro-animations, glassmorphism effects, and modern typography.
- **Dynamic Product Rendering:** Products are dynamically injected into the DOM from a structured data source.
- **Filtering & Search:** Easily find products by category (Audio, Wearables, Accessories) or use the live search bar.
- **Sorting functionality:** Sort products by popularity or price (Low to High, High to Low).
- **Interactive Shopping Cart:** 
  - Slide-out cart sidebar.
  - Add items, update quantities, or remove items.
  - Real-time subtotal calculations.
- **Data Persistence:** Cart state is saved using the browser's `localStorage` so items persist across page reloads.
- **Custom Notifications:** Toast notifications provide immediate feedback when an item is added to the cart.

## 🛠️ Technologies Used

- **HTML5:** Semantic HTML structure.
- **CSS3:** Custom variables, CSS Grid/Flexbox, transitions, and responsive design (no external frameworks like Tailwind or Bootstrap).
- **JavaScript (Vanilla):** DOM manipulation, state management, array methods (`filter`, `reduce`, `sort`), and event handling.

## 🚀 Getting Started

To run this project locally, simply clone the repository and open the `index.html` file in your browser.

```bash
# Clone the repository
git clone https://github.com/yourusername/nexus-ecommerce.git

# Navigate to the directory
cd nexus-ecommerce

# Open the project (or use a Live Server extension)
# If you have Node.js installed, you can use npx:
npx http-server -p 8080 ./
```
